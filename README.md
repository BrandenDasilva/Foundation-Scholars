# Foundation Scholars Website

A comprehensive website for the Foundation Scholars program at Western Michigan University, designed to connect current students and alumni, share news and updates, and build a vibrant networking community.

## About

The Foundation Scholars program is a prestigious scholarship initiative at Western Michigan University that recognizes and supports outstanding students. This website serves as a hub for:

- **Current Scholars**: Connect with peers, access resources, and stay informed about program activities
- **Alumni Network**: Maintain connections, mentor current scholars, and share professional experiences
- **Newsletter**: Stay updated on scholar achievements, upcoming events, and program news
- **Community Engagement**: Build lasting relationships and support each other's success

## Features

- **Responsive Design**: Mobile-friendly interface that works on all devices
- **Network Directory**: Connect with current scholars and alumni
- **Newsletter Section**: Regular updates on program news and scholar achievements
- **Contact System**: Easy way to get in touch with the program
- **Professional Design**: Western Michigan University themed colors (Brown and Gold)

## Website Structure

- `index.html` - Home page with welcome message and feature highlights
- `about.html` - Information about the Foundation Scholars program
- `network.html` - Scholar network and directory
- `newsletter.html` - Newsletter archive and subscription
- `contact.html` - Contact form and program information
- `styles/main.css` - Complete styling for the website
- `scripts/` - JavaScript for interactive features

## How to Use

This is a static website that can be hosted on any web server or platform like:

- **GitHub Pages**: Simply enable GitHub Pages in repository settings
- **Netlify**: Drag and drop deployment
- **Vercel**: Connect your repository for automatic deployments
- **Traditional Web Hosting**: Upload files via FTP

### Local Development

To view the website locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

### Deployment to GitHub Pages

1. Go to repository Settings
2. Navigate to Pages section
3. Select branch (usually `main`) and root folder
4. Save and wait for deployment
5. Your site will be available at `https://[username].github.io/Foundation-Scholars/`

## Customization

The website is designed to be easily customizable:

- **Colors**: Edit CSS variables in `styles/main.css` (`:root` section)
- **Content**: Update HTML files with your specific information
- **Images**: Add scholar photos and event images to an `images/` folder
- **Forms**: Connect contact and newsletter forms to your email service or backend
